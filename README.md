# ☁️ 云端情绪手账

> 把心事藏进云朵，与温柔不期而遇

一款温暖治愈系情绪记录 Web 应用，用云端意象承载你的每一份心情。

## ✨ 功能特色

### 📝 心事云朵
- 抽取随机云朵，记录此刻心情与文字
- 支持 7 种情绪类型选择，配以对应色彩
- 可保存为草稿或直接发布

### 📖 我的云海
- 草稿箱、收藏夹、全部手账分类浏览
- 支持收藏、编辑、删除操作
- 卡片式瀑布流展示

### 🧘 静心小筑
- 内置 3 首治愈音乐（Web Audio API 合成，带混响与和弦垫音）
- 治愈短句随机切换
- 解压泡泡小游戏

### 🏅 云途图鉴
- 情绪书写日历，可视化每日心情
- 成就徽章系统，解锁内心成长里程碑
- 统计面板：已发布、已拾取、连续天数、情绪种类

## 🎨 视觉设计

- 多层云朵动态背景 + 粒子飘浮特效
- 4 种主题色切换（晨曦粉、薄荷绿、星空紫、深夜蓝）
- 流畅的页面切换动画
- 响应式布局，适配手机与桌面

## 🛠️ 技术栈

- 纯前端实现，零依赖
- HTML5 + CSS3 + Vanilla JavaScript
- Web Audio API 合成背景音乐（混响 + 低通滤波 + 合唱效果）
- LocalStorage 本地数据持久化
